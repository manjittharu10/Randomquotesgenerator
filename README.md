# Randomquotesgenerator
A simple and interactive Random Quotes Generator built using JavaScript. This project displays a new quote each time the user clicks a button, making it a great example of dynamic content handling on the frontend.
Features:

Generates random quotes instantly

Clean and responsive UI

Easy-to-understand JavaScript logic

Uses DOM manipulation and event handling

Tech Stack:

HTML

CSS

JavaScript (Vanilla JS)

What I Learned:

Handling events in JavaScript

Updating UI dynamically using the DOM

Organizing clean and readable code

Strengthening core JavaScript concepts

Perfect for beginners who want to practice frontend fundamentals and DOM manipulation.
